# README

> Claude Code Skill: 解析数学试卷并导出为 Obsidian 格式

## 功能

1. **解析试卷**: 从 Markdown 试卷提取题目、答案、解析、知识点标签
2. **导出 Obsidian**: 生成 Obsidian 兼容的 Markdown 文件，支持双链和 Dataview
3. **学生版**: 自动生成答案隐藏的学生版试卷
4. **自动分类**: 按题型、知识点自动分类题目

## 快速开始

### 1. 放置试卷文件

将 Markdown 格式的试卷（教师版）放到此目录下，或告诉 AI 试卷的路径。

### 2. 运行解析

```bash
cd Skills/math-paper-parser
python3 math_paper_parser.py
```

### 3. 导出 Obsidian

```bash
python3 json_to_obsidian.py
```

### 4. 导入 Obsidian

将生成的 `obsidian/` 文件夹复制到你的 Obsidian 库中。

## 三、解答题

1. 解答题题干...

## 输出结构

```
obsidian/
├── index.md                  # 总索引
├── dataview-queries.md      # Dataview 查询
├── questions/               # 题目文件
│   ├── q1_xxx.md           # 第1题
│   ├── q2_xxx.md           # 第2题
│   └── ...
├── by_type/                # 按题型
│   ├── 选择题.md
│   ├── 填空题.md
│   └── 解答题.md
├── by_tag/                 # 按知识点
│   ├── 三角函数/
│   ├── 平面向量/
│   └── ...
└── student_version/        # 学生版
```

## 自动识别的知识点

| 关键词 | 标签 |
|:---|:---|
| sin/cos/tan/终边/象限角 | 三角函数, 任意角三角函数 |
| 扇形/弧长/圆心角 | 三角函数, 扇形面积与弧长 |
| 向量/数量积/共线 | 平面向量, 向量数量积 |
| 正弦定理/余三角形, 三角形弦定理 | 解性质 |
| 对称/周期/单调/极值 | 函数性质, 三角函数图象与性质 |
| 取值范围/最大值/最小值 | 不等式与最值, 参数讨论 |
