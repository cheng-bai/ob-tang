# 快速开始

## 1. 安装依赖

```bash
brew install pandoc
brew install --cask wkhtmltopdf  # 可选，渲染效果更好
```

## 2. 在 Obsidian 中使用

### 方法 A：使用模板

1. 将 `templates/obsidian-template-*.md` 复制到你的 Obsidian 模板文件夹
2. 在 Obsidian 设置中配置模板文件夹路径
3. 新建笔记时插入对应模板

### 方法 B：手动编写

按照以下格式编写题目：

```markdown
---
tags: [数学/讲义，打印]
version: student
---

# 标题

## 章节

### 例 1.1（A 组）

**题目** 题干...

- A. 选项 A
- B. 选项 B
- C. 选项 C
- D. 选项 D

> [!blank]- 留白
> - 类型：选择题
> - 难度：A
> - 行数：2

<!-- Answer
**答案** A
**解析** ...
-->
```

## 3. 导出 PDF

```bash
# 学生版（无答案）
bash scripts/render-student.sh input.md output-student.pdf

# 教师版（含答案）
bash scripts/render-teacher.sh input.md output-teacher.pdf

# 润色格式（可选）
bash scripts/polish.sh input.md
```

## 4. Obsidian 插件推荐

| 插件 | 用途 |
|------|------|
| Templater | 插入题目模板 |
| QuickAdd | 创建快捷命令 |
| Shell Commands | 直接调用导出脚本 |

### Shell Commands 配置示例

```bash
# 导出学生版
bash /path/to/markdown-print-kit/scripts/render-student.sh "{{file.path}}" "{{file.path:dirname}}/{{file.name}}-学生版.pdf"

# 导出教师版
bash /path/to/markdown-print-kit/scripts/render-teacher.sh "{{file.path}}" "{{file.path:dirname}}/{{file.name}}-教师版.pdf"
```
