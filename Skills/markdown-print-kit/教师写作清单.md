# 教师写作检查清单

编写数学讲义时的检查事项。

---

## 前置检查

- [ ] 文件命名规范：`年份-年级-数学-第X讲.md`
- [ ] Frontmatter 完整（难度、年级、课时、知识点）
- [ ] 打印版本标识正确（学生版/教师版）

## 格式检查

- [ ] 每个题目用 `<div class="question-block" markdown="1">` 包裹
- [ ] 题目编号连续
- [ ] 难度星级标注（⭐⭐⭐）
- [ ] 选择题选项格式统一（A/B/C/D）
- [ ] 留白区域使用 `> [!blank]` callout

## 内容检查

- [ ] 答案被 `<!-- Answer -->` 注释包裹
- [ ] 解析步骤完整
- [ ] LaTeX 公式渲染正确
- [ ] 图片路径正确
- [ ] 答案汇总表已生成

## 导出前检查

- [ ] 运行 `polish.sh` 润色格式
- [ ] 学生版导出测试
- [ ] 教师版导出测试
- [ ] 检查分页是否合理

---

*使用此清单确保讲义质量*
