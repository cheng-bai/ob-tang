# 函数与集合综合题集

## 一、基础题（A 组）

### 例 1.1（A 组・课后练习）

**题目** 已知集合 $A = \{x \mid -2 \leq x \leq 3\}$，$B = \{x \mid x > 1\}$，则 $A \cap B =$ （    ）

- A. $\{x \mid 1 < x \leq 3\}$
- B. $\{x \mid 1 \leq x \leq 3\}$
- C. $\{x \mid -2 \leq x \leq 3\}$
- D. $\{x \mid x > -2\}$

> [!blank]- 留白
> - 类型：选择题
> - 难度：A
> - 行数：2

<!-- Answer
**答案** A
**解析** $A \cap B$ 表示同时属于 $A$ 和 $B$ 的元素。$A$ 中 $x$ 的范围是 $[-2, 3]$，$B$ 中 $x$ 的范围是 $(1, +\infty)$，取交集得 $(1, 3]$，即 $\{x \mid 1 < x \leq 3\}$。
-->

### 例 1.2（A 组・改编）

**题目** 函数 $f(x) = \sqrt{x-1} + \dfrac{1}{x-2}$ 的定义域是（    ）

- A. $[1, 2) \cup (2, +\infty)$
- B. $(1, 2) \cup (2, +\infty)$
- C. $[1, +\infty)$
- D. $(2, +\infty)$

> [!blank]- 留白
> - 类型：选择题
> - 难度：A
> - 行数：2

<!-- Answer
**答案** A
**解析** 根号下需满足 $x-1 \geq 0 \Rightarrow x \geq 1$；分母需满足 $x-2 \neq 0 \Rightarrow x \neq 2$。综合得 $x \in [1, 2) \cup (2, +\infty)$。
-->

## 二、中档题（B 组）

### 例 2.1（B 组・期中真题）

**题目** 已知函数 $f(x) = x^2 - 2ax + 3$ 在区间 $[1, 3]$ 上单调递增，则实数 $a$ 的取值范围是________。

> [!blank]- 留白
> - 类型：填空题
> - 难度：B
> - 行数：3

<!-- Answer
**答案** $a \leq 1$
**解析** $f(x)$ 的对称轴为 $x = a$。开口向上的二次函数在对称轴右侧单调递增，故需 $[1, 3]$ 在对称轴右侧，即 $a \leq 1$。
-->

### 例 2.2（B 组・改编）

**题目** 设函数 $f(x) = \dfrac{ax + 1}{x + 2}$ 在 $(-2, +\infty)$ 上单调递增，求实数 $a$ 的取值范围。

> [!blank]- 留白
> - 类型：解答题
> - 难度：B
> - 行数：6

<!-- Answer
**答案** $a > \dfrac{1}{2}$
**解析** 
方法一（导数法）：
$$f'(x) = \frac{a(x+2) - (ax+1)}{(x+2)^2} = \frac{2a-1}{(x+2)^2}$$
由 $f'(x) > 0$ 得 $2a - 1 > 0$，即 $a > \dfrac{1}{2}$。

方法二（定义法）：任取 $x_1 < x_2$，计算 $f(x_1) - f(x_2)$ 并分析符号。

## 三、提高题（C 组）

### 例 3.1（C 组・竞赛改编）

**题目** 已知函数 $f(x)$ 满足 $f(x+y) = f(x) + f(y) + 2xy$ 对任意实数 $x, y$ 成立，且 $f(1) = 2$。

(1) 求 $f(0)$ 的值；

(2) 求 $f(x)$ 的解析式；

(3) 若不等式 $f(x) \geq kx$ 对任意 $x \in [0, 1]$ 恒成立，求实数 $k$ 的取值范围。

> [!blank]- 留白
> - 类型：解答题
> - 难度：C
> - 行数：10

<!-- Answer
**答案** (1) $f(0) = 0$；(2) $f(x) = x^2 + x$；(3) $k \leq 2$

**解析** 
(1) 令 $x = y = 0$，得 $f(0) = 2f(0) \Rightarrow f(0) = 0$。

(2) 令 $y = 1$，得 $f(x+1) = f(x) + f(1) + 2x = f(x) + 2 + 2x$。
猜测 $f(x) = x^2 + x$，验证：
$$f(x+y) = (x+y)^2 + (x+y) = x^2 + y^2 + 2xy + x + y$$
$$f(x) + f(y) + 2xy = (x^2 + x) + (y^2 + y) + 2xy$$
两式相等，且 $f(1) = 1 + 1 = 2$，符合题意。

(3) 由 $f(x) \geq kx$ 得 $x^2 + x \geq kx$，即 $x^2 + (1-k)x \geq 0$。
当 $x = 0$ 时恒成立；当 $x \in (0, 1]$ 时，需 $x + (1-k) \geq 0 \Rightarrow k \leq x + 1$。
$(x+1)_{\min} = 1$（当 $x \to 0^+$），但需考虑端点 $x=1$ 时 $k \leq 2$。
综上，$k \leq 2$。
-->
