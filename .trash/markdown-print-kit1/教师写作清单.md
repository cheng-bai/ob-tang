# 教师写作清单

## 格式检查

- [ ] 文档有 frontmatter（tags, created, version）
- [ ] 标题层级正确：`#` → `##` → `###`
- [ ] 每个 `###` 下只有一个题目
- [ ] 选择题选项是连续四行 `- A./- B./- C./- D.`
- [ ] 每个题目后有 `> [!blank]` 留白块
- [ ] 答案放在 `<!-- Answer ... -->` 注释中

## 留白速查

| 题型 | A 组 | B 组 | C 组 |
|------|------|------|------|
| 选择题 | `lines=2` | `lines=3` | `lines=3` |
| 填空题 | `lines=2` | `lines=3` | `lines=4` |
| 解答题 | `lines=6` | `lines=8` | `lines=10` |

## 快速模板

### 选择题

```markdown
### 例 X.X（A 组・来源）

**题目** 题干...

- A. 选项 A
- B. 选项 B
- C. 选项 C
- D. 选项 D

> [!blank]- 留白
> - 类型：选择题
> - 难度：A
> - 行数：2

<!-- Answer
**答案** A
**解析** ...
-->
```

### 填空题

```markdown
### 例 X.X（B 组・来源）

**题目** 题干...，结果是________。

> [!blank]- 留白
> - 类型：填空题
> - 难度：B
> - 行数：3

<!-- Answer
**答案** ...
**解析** ...
-->
```

### 解答题

```markdown
### 例 X.X（C 组・来源）

**题目** 题干...

(1) 第一问...
(2) 第二问...

> [!blank]- 留白
> - 类型：解答题
> - 难度：C
> - 行数：10

<!-- Answer
**答案** ...
**解析** ...
-->
```

## 导出命令

```bash
# 学生版
bash scripts/render-student.sh input.md output-student.pdf

# 教师版
bash scripts/render-teacher.sh input.md output-teacher.pdf

# 润色格式
bash scripts/polish.sh input.md
```

## 注意事项

1. 题块内部不要插入多余空行
2. 长题优先拆成 `(1)(2)(3)` 小问
3. 公式使用 `$...$` 或 `$$...$$`
4. 答案注释不要跨题目
