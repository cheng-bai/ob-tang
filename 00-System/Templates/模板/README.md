## 使用说明

1. **复制模板**：将模板复制到目标目录
2. **使用 Templater**：配置 Templater 插件实现日期、标题等自动填充
3. **题块语法**：
   - `{{template:填空题}}` - 填空题题块
   - `{{template:选择题}}` - 选择题题块
   - `{{template:解答题}}` - 解答题题块
   - `{{template:例题}}` - 例题题块
   - `{{template:练习题}}` - 练习题题块
   - `{{template:高考真题}}` - 高考真题题块
