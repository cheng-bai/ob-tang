# q21_已知函数 f( xrigh

## 题目

已知函数 $f\left( x\right)  = \sin \frac{\pi x}{2}\left( {x \in  R}\right)$ . 任取 $t \in  R$ ,若函数 $y = f\left( x\right)$ 在区间 $\left\lbrack  {t, t + 1}\right\rbrack$ 上的最大值为 $M\left( t\right)$ ,最小值为 $m\left( t\right)$ ,记 $g\left( t\right)  = M\left( t\right)  - m\left( t\right)$ .
(1)求函数 $y = f\left( x\right)$ 的最小正周期及对称轴方程；
( 2 )当 $t \in  \left\lbrack  {-2,0}\right\rbrack$ 时，求函数 $y = g\left( t\right)$ 的解析式；
(3)设函数 $h\left( x\right)  = {2}^{\left| x - k\right| }, H\left( x\right)  = x\left| {x - k}\right|  + {2k} - 8$ ，其中 $k$ 为实数，且满足关于 $t$ 的不等式 $\sqrt{2}k - {4g}\left( t\right)  \leq  0$ 有解,若对任意 ${x}_{1} \in  \lbrack 4, + \infty )$ ,存在 ${x}_{2} \in  ( - \infty ,4\rbrack$ ,使得 $h\left( {x}_{2}\right)  = H\left( {x}_{1}\right)$ 成立,求实数 $k$ 的取值范围.

## 解析

(1) 函数 $f\left( x\right)  = \sin \frac{\pi x}{2}\left( {x \in  R}\right)$ ,则 $f\left( x\right)$ 的最小正周期为 $T = \frac{2\pi }{\frac{\pi }{2}} = 4$ ; 令 $\frac{\pi }{2}x = {k\pi } + \frac{\pi }{2}$ ,解得 $f\left( x\right)$ 的对称轴方程为 $x = {2k} + 1\left( {x \in  Z}\right)$ ;
(2)①当 $t \in  \left\lbrack  {-2, - \frac{3}{2}}\right)$ 时，在区间 $\left\lbrack  {t, t + 1}\right\rbrack$ 上，
$M\left( t\right)  = f\left( t\right)  = \sin \frac{\pi t}{2},\;m\left( t\right)  = f\left( {-1}\right)  =  - 1,$
所以 $g\left( t\right)  = M\left( t\right)  - m\left( t\right)  = 1 + \sin \frac{\pi t}{2}$ ;
② 当 $t \in  \left\lbrack  {-\frac{3}{2}, - 1}\right)$ 时，在区间 $\left\lbrack  {t, t + 1}\right\rbrack$ 上，
$M\left( t\right)  = f\left( {t + 1}\right)  = \sin \left\lbrack  {\frac{\pi }{2}\left( {t + 1}\right) }\right\rbrack   = \cos \frac{\pi t}{2},\;m\left( t\right)  = f\left( {-1}\right)  =  - 1,$
所以 $g\left( t\right)  = M\left( t\right)  - m\left( t\right)  = 1 + \cos \frac{\pi t}{2}$ ;
③ 当 $t \in  \left\lbrack  {-1,0}\right\rbrack$ 时，在区间 $\left\lbrack  {t, t + 1}\right\rbrack$ 上，
$M\left( t\right)  = f\left( {t + 1}\right)  = \sin \left\lbrack  {\frac{\pi }{2}\left( {t + 1}\right) }\right\rbrack   = \cos \frac{\pi t}{2},\;m\left( t\right)  = f\left( t\right)  = \sin \frac{\pi t}{2},$
所以 $g\left( t\right)  = M\left( t\right)  - m\left( t\right)  = \cos \frac{\pi t}{2} - \sin \frac{\pi t}{2}$ ;
所以当 $t \in  \left\lbrack  {-2,0}\right\rbrack$ 时,函数 $g\left( t\right)  = \left\{  \begin{array}{l} \cos \frac{\pi t}{2} - \sin \frac{\pi t}{2}, t \in  \left\lbrack  {-1,0}\right\rbrack  \\  1 + \cos \frac{\pi t}{2}, t \in  \left\lbrack  {-\frac{3}{2}, - 1}\right) \\  1 + \sin \frac{\pi t}{2}, t \in  \left\lbrack  {-2, - \frac{3}{2}}\right)  \end{array}\right.$
(3) 因为 $f\left( x\right)  = \sin \frac{\pi x}{2}$ 的最小正周期 $T = 4$ ,
所以 $M\left( {t + 4}\right)  = M\left( t\right) , m\left( {t + 4}\right)  = m\left( t\right)$ ,
所以 $g\left( {t + 4}\right)  = M\left( {t + 4}\right)  - m\left( {t + 4}\right)  = M\left( t\right)  - m\left( t\right)  = g\left( t\right)$ ,
所以 $g\left( t\right)$ 是周期为 4 的函数,研究函数 $g\left( t\right)$ 的性质,
只须研究函数 $g\left( t\right)$ 在 $t \in  \left\lbrack  {-2,2}\right\rbrack$ 时的性质即可;
仿照 (2),得 $g\left( t\right)  = \left\{  \begin{array}{l} 1 + \sin \frac{\pi t}{2}, t \in  \lbrack  - 2, - \frac{3}{2}) \\  1 + \cos \frac{\pi t}{2}, t \in  \lbrack  - \frac{3}{2}, - 1) \\  \cos \frac{\pi t}{2} - \sin \frac{\pi t}{2}, t \in  \lbrack  - 1,0) \\  1 - \sin \frac{\pi t}{2}, t \in  \lbrack 0,\frac{1}{2}) \\  1 - \cos \frac{\pi t}{2}, t \in  \lbrack \frac{1}{2},1) \\  \sin \frac{\pi t}{2} - \cos \frac{\pi t}{2}, t \in  \left\lbrack  {1,2}\right\rbrack   ![bo_d6alfsf7aajc739aqkp0_12_458_773_669_359_0.jpg](00-System/06-工具脚本/images/bo_d6alfsf7aajc739aqkp0_12_458_773_669_359_0.jpg)8_773_669_359_0.jpg)

所以函数 $g\left( t\right)$ 的值域为 $\left\lbrack  {1 - \frac{\sqrt{2}}{2},\sqrt{2}}\right\rbrack$ ;
已知 $\sqrt{2}k - {4g}\left( t\right)  \leq  0$ 有解，即 $\sqrt{2}k \leq  {4g}{\left( t\right) }_{\max } = 4\sqrt{2}$ ，所以 $k \leq  4$ ； 若对任意 ${x}_{1} \in  \lbrack 4, + \infty )$ ,存在 ${x}_{2} \in  ( - \infty ,4\rbrack$ ,使得 $h\left( {x}_{2}\right)  = H\left( {x}_{1}\right)$ 成立, 即 $H\left( x\right)$ 在 $\lbrack 4, + \infty )$ 的值域是 $h\left( x\right)$ 在 $( - \infty ,4\rbrack$ 的值域的子集.
因为 $h\left( x\right)  = {2}^{\left| x - k\right| } = \left\{  \begin{array}{l} {2}^{x - k}\left( {x \geq  k}\right) \\  {2}^{k - x}\left( {x < k}\right)  \end{array}\right.$ ,
当 $k \leq  4$ 时,因为 $h\left( x\right)$ 在 $\left( {-\infty , k}\right)$ 上严格减,在 $\left\lbrack  {k,4}\right\rbrack$ 上严格增,
所以 $h{\left( x\right) }_{\min } = h\left( k\right)  = 1$ ,
因为 $H\left( x\right)  = x\left| {x - k}\right|  + {2k} - 8$ 在 $\lbrack 4, + \infty )$ 上严格增,
所以 $H{\left( x\right) }_{\min } = H\left( 4\right)  = 8 - {2k}$ ,所以 $8 - {2k} \geq  1$ ,即 $k \leq  \frac{7}{2}$ ;
当 $4 < k \leq  5$ 时,因为函数 $h\left( x\right)$ 在 $( - \infty ,4\rbrack$ 上严格减,
所以 $h{\left( x\right) }_{\min } = h\left( 4\right)  = {2}^{k - 4}$ ,
$H\left( x\right)  = x\left| {x - k}\right|  + {2k} - 8 = \left\{  \begin{array}{l} {x}^{2} - {kx} + {2k} - 8, x \geq  k \\   - {x}^{2} + {kx} + {2k} - 8,4 \leq  x < k \end{array}\right.$ ,
所以,函数 $H\left( x\right)$ 在 $\lbrack 4, k)$ 上严格减,在 $\lbrack k, + \infty )$ 上严格增,
