# q14_在 bigtriangleup

## 题目

在 $\bigtriangleup  {ABC}$ 中，设 $1 - \cos A\cos B = {\cos }^{2}\frac{C}{2}$ ，则 $\bigtriangleup  {ABC}$ 的形状为 ( )
A. 直角三角形 B. 锐角三角形 C. 等腰三角形 D. 钝角三角形

### 选项

- **A.** 直角三角形 B. 锐角三角形 C. 等腰三角形 D. 钝角三角形

## 解析

因为 $1 - \cos A\cos B = {\cos }^{2}\frac{C}{2}$ ,所以 $1 - \cos A\cos B = \frac{1 + \cos C}{2}$ ,
化简整理得 $1 - 2\cos A\cos B = \cos C$ ,因为 $A + B + C = \pi$ ,
所以 $\cos C = \cos \left\lbrack  {\pi  - \left( {A + B}\right) }\right\rbrack   =  - \cos \left( {A + B}\right)  =  - \cos A\cos B + \sin A\sin B$ ,
所以 $1 - 2\cos A\cos B =  - \cos A\cos B + \sin A\sin B$ ,
所以 $1 = \cos A\cos B + \sin A\sin B = \cos \left( {A - B}\right)$ ,
又因为 $A\text{ 、 }B \in  \left( {0,\pi }\right)$ ,所以 $A - B = 0$ ,即 $A = B$ ,所以 $\bigtriangleup {ABC}$ 为等腰三角形.
故选 $C$ .
