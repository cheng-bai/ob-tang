# 解题步骤规范

> AI 辅助解题时的标准步骤规范

## 标准化解题流程

### 1. 审题分析
- 明确题目类型（选择/填空/解答）
- 识别已知条件和求解目标
- 确定涉及的知识点和考点
- 评估题目难度

### 2. 思路分析
- 回顾相关知识点和公式
- 寻找解题突破口
- 选择合适的方法（数形结合、分类讨论、换元等）
- 制定解题计划

### 3. 规范解答
- 步骤完整，逻辑清晰
- 关键步骤写出依据
- 计算过程规范
- 最终答案明确

### 4. 答案检验
- 检查计算结果
- 验证答案合理性
- 一题多解可作比较

### 5. 反思总结
- 提炼解题方法
- 指出易错点
- 总结相关题型

---

## 各类题目规范

### 选择题
1. 直接法：直接计算求解
2. 排除法：排除错误选项
3. 特殊值法：代入验证
4. 数形结合：借助图像分析

### 填空题
1. 结果化简到最简形式
2. 注意定义域限制
3. 答案书写规范

### 解答题
1. 格式规范：解、证明开头
2. 步骤完整：每步有依据
3. 分类讨论：不重不漏
4. 书写整洁：便于阅卷

## 常见题型解题要点

| 题型 | 关键步骤 | 注意事项 |
|:---|:---|:---|
| 函数零点 | 1. 画图像 2. 找交点 | 定义域优先 |
| 导数综合 | 1. 求导 2. 研单调 3. 极值最值 | 分类讨论 |
| 数列求和 | 1. 判断类型 2. 错位/裂项/分组 | 公式准确 |
| 解析几何 | 1. 设方程 2. 代坐标 3. 化简 | 运算仔细 |
| 立体几何 | 1. 证垂直 2. 建坐标 3. 求距离 | 方向余弦 |
