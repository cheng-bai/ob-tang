【解析】 $\overrightarrow{a},\overrightarrow{b}$ 是两个不共线的向量,所以 $\frac{2}{3}\overrightarrow{a} - \frac{1}{3}\overrightarrow{b} \neq  \overrightarrow{0}$ ,

因为 $\overrightarrow{b} + t\overrightarrow{a},\frac{2}{3}\overrightarrow{a} - \frac{1}{3}\overrightarrow{b}$ 共线,所以 $\overrightarrow{b} + t\overrightarrow{a} = \lambda \left( {\frac{2}{3}\overrightarrow{a} - \frac{1}{3}\overrightarrow{b}}\right) ,\lambda  \in  R$ ,

所以 $\left( {t - \frac{2}{3}\lambda }\right) \overrightarrow{a} + \left( {1 + \frac{1}{3}\lambda }\right) \overrightarrow{b} = \overrightarrow{0}$ ,所以 $t - \frac{2}{3}\lambda  = 0$ 且 $1 + \frac{1}{3}\lambda  = 0$ ,

14. 在 $\bigtriangleup  {ABC}$ 中，设 $1 - \cos A\cos B = {\cos }^{2}\frac{C}{2}$ ，则 $\bigtriangleup  {ABC}$ 的形状为 ( )

A. 直角三角形 B. 锐角三角形 C. 等腰三角形 D. 钝角三角形

【解析】因为 $1 - \cos A\cos B = {\cos }^{2}\frac{C}{2}$ ,所以 $1 - \cos A\cos B = \frac{1 + \cos C}{2}$ ,

化简整理得 $1 - 2\cos A\cos B = \cos C$ ,因为 $A + B + C = \pi$ ,

所以 $\cos C = \cos \left\lbrack  {\pi  - \left( {A + B}\right) }\right\rbrack   =  - \cos \left( {A + B}\right)  =  - \cos A\cos B + \sin A\sin B$ ,

所以 $1 - 2\cos A\cos B =  - \cos A\cos B + \sin A\sin B$ ,

所以 $1 = \cos A\cos B + \sin A\sin B = \cos \left( {A - B}\right)$ ,

又因为 $A\text{ 、 }B \in  \left( {0,\pi }\right)$ ,所以 $A - B = 0$ ,即 $A = B$ ,所以 $\bigtriangleup {ABC}$ 为等腰三角形.

16. 关于函数 $f\left( x\right)  = \frac{\sqrt{2}\sin \left| {\frac{\pi }{2}x}\right| }{{x}^{2} - {2x} + 3}$ 的以下两个命题:

①函数 $y = f\left( x\right)$ 的图像是轴对称图形；

②对任意的 $x \in  R$ ，不等式 $2\left| {f\left( x\right) }\right|  \leq  3\left| x\right|$ 恒成立.

则正确的是( )

A. ①正确②正确 B. ①正确②错误 C. ①错误②正确

即函数 $y = f\left( x\right)$ 的图像是轴对称图形,所以命题①正确.

已知 $f\left( x\right)  = \frac{\sqrt{2}\sin \left| {\frac{\pi }{2}x}\right| }{{x}^{2} - {2x} + 3}$ ,则 $2\left| {f\left( x\right) }\right|  = \frac{2\sqrt{2}\left| \sin \right| \frac{\pi }{2}x\parallel }{{x}^{2} - {2x} + 3}$ .

18. 已知向量 $\overrightarrow{a}$ 、 $\overrightarrow{b}$ 满足 $\left| \overrightarrow{a}\right|  = \sqrt{3}$ ， $\left| \overrightarrow{b}\right|  = 1$ ，设 $\overrightarrow{a}$ 与 $\overrightarrow{b}$ 的夹角为 $\theta$ .

(2)不等式 $\left| {\overrightarrow{a} + x\overrightarrow{b}}\right|  \geq  \left| {\overrightarrow{a} + \overrightarrow{b}}\right|$ 恒成立，
