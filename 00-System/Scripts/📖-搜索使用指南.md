# 📖-搜索使用指南

> 本指南帮助您更高效地使用 Obsidian 搜索功能

---

## 快速搜索快捷键

| 操作 | 快捷键 | 说明 |
|:---|:---|:---|
| 全局搜索 | `Ctrl + Shift + F` | 在所有文件中搜索 |
| Omnisearch | `Ctrl + O` | 快速搜索（需配置） |
| 搜索当前文件 | `Ctrl + F` | 在当前文件中搜索 |
| 标签搜索 | `#` | 输入 # 开启标签搜索 |

---

## 搜索技巧

### 1. 精确匹配
```
"精确短语"
```
搜索完全匹配的短语

### 2. 文件名搜索
```
file:试卷
```
搜索文件名包含"试卷"的文件

### 3. 标签搜索
```
tag:函数
```
搜索包含 #函数 标签的文件

### 4. 路径搜索
```
path:01-数学教学
```
在指定路径下搜索

### 5. 组合搜索
```
tag:高三 path:讲义 函数
```
搜索高三讲义中关于函数的文件

---

## Omnisearch 增强功能

本 vault 已配置 Omnisearch 插件，支持：

- **模糊搜索**：输入部分关键词也能匹配
- **智能排序**：最近编辑的文件会优先显示
- **中文分词**：支持中文搜索
- **标题权重**：文件标题匹配会排在前面

## 推荐工作流

1. **按 Ctrl + O** 打开 Omnisearch
2. **输入关键词**（如"函数"）
3. **使用方向键** 选择结果
4. **按 Enter** 打开文件
5. **使用 Ctrl+Shift+F** 进行深度搜索
