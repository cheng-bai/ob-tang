# q8_已知 atext 、 beta

## 题目

已知 $a\text{ 、 }\beta$ 均为锐角, $\sin a = 3\sin \beta \cos \left( {a + \beta }\right)$ ,则 $\tan a$ 取得最大值时, $\tan \left( {a + \beta }\right)$ 的值为___.

## 解析

已知 $a\text{ 、 }\beta$ 均为锐角, $\sin a = 3\sin \beta \cos \left( {a + \beta }\right)$ ,
则 $\sin \left\lbrack  {\left( {a + \beta }\right)  - \beta }\right\rbrack   = 3\sin \beta \cos \left( {a + \beta }\right)$ ,
即 $\sin \left( {a + \beta }\right) \cos \beta  - \cos \left( {a + \beta }\right) \sin \beta  = 3\sin \beta \cos \left( {a + \beta }\right)$ ,
即 $\sin \left( {a + \beta }\right) \cos \beta  = 4\sin \beta \cos \left( {a + \beta }\right)$ ,即 $\tan \left( {a + \beta }\right)  = 4\tan \beta  > 0$ ,
则 $\tan a = \frac{\tan \left( {a + \beta }\right)  - \tan \beta }{1 + \tan \left( {a + \beta }\right) \tan \beta } = \frac{\frac{3}{4}\tan \left( {a + \beta }\right) }{1 + \frac{{\tan }^{2}\left( {a + \beta }\right) }{4}} = \frac{3}{\tan \left( {a + \beta }\right)  + \frac{4}{\tan \left( {a + \beta }\right) }}$
$\leq  \frac{3}{2\sqrt{\tan \left( {a + \beta }\right)  \times  \frac{4}{\tan \left( {a + \beta }\right) }}} = \frac{3}{4},$
当且仅当 $\tan \left( {a + \beta }\right)  = \frac{4}{\tan \left( {a + \beta }\right) }$ ,即 $\tan \left( {a + \beta }\right)  = 2$ 时取等号,
则 $\tan a$ 取得最大值时, $\tan \left( {a + \beta }\right)$ 的值为 2 .
