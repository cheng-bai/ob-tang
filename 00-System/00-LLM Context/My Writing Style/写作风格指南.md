# 写作风格指南

## 语言特点
- 语气：专业但亲和，避免过于学术化
- 句式偏好：长短句交替，关键结论用短句
- 常用比喻/类比：用生活化的例子解释数学概念

## 内容风格
- 开头方式：直接点明知识点或题目核心
- 结尾方式：总结关键方法或易错点
- 段落过渡方式：使用"接下来""值得注意的是"等过渡词

## 用词偏好
- 常用词汇：
  - 数学术语：函数、零点、极值、导数等
  - 教学用语：掌握、理解、运用、迁移
- 避免词汇：过于口语化的表达
- 专业术语使用：准确、规范、符合教材表述

## 格式偏好
- 标题格式：## 章节标题，### 小节标题
- 列表格式：使用编号和项目符号
- 强调方式：**加粗**或 > 引用

## 参考风格
- 喜欢的风格：清晰有条理，重点突出
- 风格描述：解题步骤详细但不冗余，方法总结精炼到位
