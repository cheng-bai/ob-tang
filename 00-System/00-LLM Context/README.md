# README

> 本目录包含承白老师使用 AI 辅助教学时的上下文配置

---

## 目录结构

```
00-LLM Context/
├── Personal Profile/
│   └── 个人简介.md       # 基本信息、教学特点
├── Basic Rules/
│   └── 使用规则.md        # 交互原则、任务优先级
├── My Writing Style/
│   └── 写作风格指南.md    # 语言特点、内容风格
├── My Flavor/
│   └── 个人偏好.md        # 个人习惯、偏好
├── Teaching Rules/
│   ├── 解题步骤规范.md    # 标准解题流程
│   └── 教学风格偏好.md    # 教学理念、讲解风格
└── Dynamic Activities/
    └── 近期动态.md        # 当前教学进度、计划
```

---

## 使用说明

### 1. 个人简介
- 包含姓名、身份、教学特点
- AI 了解用户背景的基础

### 2. 使用规则
- 交互原则、沟通方式
- 任务优先级、限制说明

### 3. 写作风格指南
- 语言特点、内容风格
- 格式偏好、参考风格

### 4. 教学规则
- **解题步骤规范**：标准解题流程
- **教学风格偏好**：讲解方式、输出格式

### 5. 近期动态
- 当前教学进度
- 近期考试/活动安排
- 需要关注的重点

---

## AI 调用优先级

1. 首先读取 **个人简介**
2. 参考 **使用规则** 确定交互方式
3. 遵循 **写作风格指南** 保持一致性
4. 应用 **解题步骤规范** 进行题目讲解
5. 参考 **教学风格偏好** 调整讲解方式

## 常用提示词模板

### 请求解题
```
请按照标准解题步骤解答此题，并给出思路分析
```

### 请求出题
```
请根据以下知识点出5道选择题，难度适中
```

### 请求讲义
```
请帮我设计一节复习课讲义，包含知识清单和练习题
```

### 请求知识总结
```
请帮我总结[知识点]的核心内容和常见题型
```
