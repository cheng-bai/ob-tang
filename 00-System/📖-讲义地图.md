## 📚 按专题分类

### 函数专题
```dataview
TABLE 
  grade as "年级",
  semester as "学期",
  date as "日期"
FROM "01零散资料"
WHERE contains(subject, "函数") OR contains(file.name, "函数")
SORT date DESC
LIMIT 10
```

### 解析几何专题
```dataview
TABLE 
  grade as "年级",
  semester as "学期",
  date as "日期"
FROM "01零散资料"
WHERE contains(subject, "直线") OR contains(subject, "圆") OR contains(subject, "圆锥曲线")
SORT date DESC
LIMIT 10
```

### 数列专题
```dataview
TABLE 
  grade as "年级",
  semester as "学期",
  date as "日期"
FROM "01零散资料"
WHERE contains(subject, "数列")
SORT date DESC
LIMIT 10
```
