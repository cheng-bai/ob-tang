## 📊 题目统计

```dataview
TABLE WITHOUT ID
  "📚 总计: " + length(rows) + " 道题目" as "统计"
FROM "01零散资料" OR "04-试卷汇总"
WHERE file.name != null
GROUP BY "题目总数"
```

## 🔢 按知识点分类

### 集合与函数
```dataview
TABLE WITHOUT ID
  file.link as "题目",
  difficulty as "难度",
  source as "来源",
  year as "年份"
FROM "01零散资料" OR "04-试卷汇总"
WHERE contains(knowledge_points, "函数") OR contains(tags, "#数学/集合与函数")

LIMIT 20
```
