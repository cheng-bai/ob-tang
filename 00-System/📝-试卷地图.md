## 🔍 Dataview 查询

### 所有试卷列表
```dataview
TABLE 
  year as "年份",
  exam_type as "类型",
  region as "地区",
  date as "日期"
FROM "04-试卷汇总"
WHERE file.name != "10-试卷汇总"
SORT year DESC, date DESC
```

### 按类型查询
```dataview
TABLE 
  year as "年份",
  region as "地区"
FROM "04-试卷汇总"
WHERE exam_type = "高考真题"
SORT year DESC
```

### 按年份查询
```dataview
TABLE 
  exam_type as "类型",
  region as "地区"
FROM "04-试卷汇总"
WHERE year = "2024"
SORT exam_type
```

## 📈 试卷分析

### 知识点覆盖统计
```dataview
TABLE WITHOUT ID
  knowledge_point as "知识点",
  count as "出现次数"
FROM "04-试卷汇总"
FLATTEN knowledge_points as knowledge_point
GROUP BY knowledge_point
SORT count DESC
LIMIT 15
```

### 难度分布
```dataview
TABLE WITHOUT ID
  difficulty as "难度",
  count as "数量"
FROM "04-试卷汇总"
GROUP BY difficulty
SORT difficulty
```
