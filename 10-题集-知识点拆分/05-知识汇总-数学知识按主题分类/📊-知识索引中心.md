## 📚 题目索引（按知识点）

### 集合与函数

```dataview
TABLE WITHOUT ID
  file.link as "题目",
  难度,
  来源,
  年份
FROM "01零散资料" OR "04-试卷汇总"
WHERE contains(tags, "题目收集") OR contains(知识点, "函数")
```

## 🏷️ 标签统计

```dataview
TABLE WITHOUT ID
  tag as "标签",
  length(rows) as "数量"
FROM "01零散资料" OR "04-试卷汇总"
FLATTEN tags as tag
WHERE tag != null
GROUP BY tag
SORT length(rows) DESC
LIMIT 20
```

## 📊 资源统计仪表盘

### 文件类型分布
```dataview
TABLE WITHOUT ID
  "📚" + length(filter(rows, f => f.type = "讲义")) as "讲义",
  "📝" + length(filter(rows, f => f.type = "试卷")) as "试卷",
  "🎯" + length(filter(rows, f => f.type = "题目")) as "题目",
  "📖" + length(filter(rows, f => f.type = "知识点")) as "知识点"
FROM "01零散资料" OR "04-试卷汇总" OR "10-题集-知识点拆分"
GROUP BY "统计"
```

### 按年份统计
```dataview
TABLE WITHOUT ID
  year as "年份",
  length(rows) as "文件数量"
FROM "01零散资料" OR "04-试卷汇总"
WHERE year != null
GROUP BY year
SORT year DESC
LIMIT 10
```
