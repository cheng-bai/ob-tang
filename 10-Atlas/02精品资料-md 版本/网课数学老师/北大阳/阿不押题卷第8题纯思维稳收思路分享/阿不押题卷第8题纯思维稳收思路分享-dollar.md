# 阿不押题卷第8题纯思维稳收思路分享-dollar
题目(命题人:阿不高中数学) 已知椭圆 $\frac{{x}^{2}}{{a}^{2}} + \frac{{y}^{2}}{{b}^{2}} = 1\left( {a > b > 0}\right)$ 左顶点为 $A$ ，点 $B, C$ 是椭圆上两点，分别位于 $y$ 轴的上方和下方，满足 ${AB}\bot {AC}$ ，且 $\sin \angle {ABC} = \frac{1}{3}$ ，则直线 ${AB}$ 斜率的取值范围是___.

读完题, 会发现这是一个大小无关的问题, 把椭圆保形状的放大缩小不会影响整个问题。椭圆本身被两个参数的控制,也就是有两个自由度, $\mathrm{B}$ 和 $\mathrm{C}$ 是椭圆上的两个点,相当于二者新添加两个自由度, 但是正好两个 “等式” 的已知把它们牵制住了, 所以椭圆定了那么 B, C 也就得能解出来,所以问题实际自由度还是 2,再配合大小无关,把大小这一个自由度抹掉，所以问题的本质是一个单自由度的问题。

我们再来重点关注它的要素是如何生成的。既然大小无关, 那就是单参数的形状指标先控制椭圆,然后椭圆上放两个点配两个牵制条件确定 B, C , 所以实际上就是椭圆的形状指标控制了 B, C 的全部信息。最后要计算一个斜率,和 B 相关的。所以目前的想法就很清楚自然了，试试看把这个 AB 的斜率用椭圆的某个形状指标(离心率，边长比例等)表示出来,直接看范围即可。这是最原始的要素关系网使用思路,完全不需要任何思维灵感。

大致的方法选定了之后，我们来尝试计算。首先，用椭圆的形状指标去算斜率，并不需要强行去先找一个这样的指标,我们就正常用 $\frac{{x}^{2}}{{a}^{2}} + \frac{{y}^{2}}{{b}^{2}} = 1$ 计算即可。因为问题全是大小无关的, 所以算到最后必定是齐次的。

我们来尝试解一下 B 和 C 。已知条件全是关于 AB 与 AC 的, 所以自然把思维焦点放到它们上面去。垂直条件如何充要替换? 那要么就是向量要么就是斜率, 这里还是选斜率好一些, 毕竟求的就是斜率。另一个角度呢? 直接计算那个角度并不方便。把条件全标在图里, 画画图看看怎么能造出那个角度? 拿个垂直尺子转一转看看, 很容易就看出两个条件就是在说 $\bigtriangleup  \mathrm{{ABC}}$ 是个固定形状的玩意。如果我让你在图中尝试找出正好是那个角的对应位置, 很明显, 直接看这个三角形的形状对没对上即可。那看法就很多了。在这个图里直接看边的比例 $\frac{\left| \mathrm{{AB}}\right| }{\left| \mathrm{{AC}}\right| }$ 是最舒服的,找的时候量边长肯定比量那个歪角舒服,最关键的是, 这两个边在有了斜率之后立刻能被直接控制。我们之前讲过解点的问题, 当直线和圆锥曲线交于两个点, 并且其中一个点还已知的时候, 另一个点用韦达可以秒出结果。这下直接把方程的列法都看透了, 大致的解题思路预判结束, 可以直接上手开始计算了。

因为大小无关,不妨直接设 $\mathrm{a} = 1$ ,这样少写一个字母 (注意,当且仅当大小无关时才可以随便设定一个长度)。再设 ${AB} : x = {ty} - 1\left( {t > 0}\right)$ ,这里用 $t$ 作为控制 $B$ 的参数,而没直接写斜率，其实无所谓。只要你是选了正确数量和位置的参数作为主自由度，那么整个要素关系网就握在了手中，最后要求什么其实不重要，只是一个从关系网向外输出的过程罢了。这时候椭圆变成了 ${x}^{2} + \frac{{y}^{2}}{{b}^{2}} = 1$ ,代入 $x = {ty} - 1$ 化简得 $\left( {{t}^{2} + \frac{1}{{b}^{2}}}\right) {y}^{2} - {2ty} = 0$ ,这里不如再把 $\frac{1}{{b}^{2}}$ 换成 $u > 1$ (这种在不受干扰的前提下换个字母的习惯大家可以多学习，注意好定义域即可),于是得 ${y}_{B} = \frac{2t}{{t}^{2} + u}$ 。然后弦长 $\left| {AB}\right|  = \sqrt{1 + {t}^{2}}\left| {{y}_{B} - {y}_{A}}\right|  = \frac{2\left| t\right| \sqrt{1 + {t}^{2}}}{{t}^{2} + u}$ 。

另一个弦长 $\left| \mathrm{{AC}}\right|$ 无非就是刚才的 $\mathrm{t}$ 换成 $- \frac{1}{\mathrm{t}}$ 然后重复刚才的计算,所以把 $\mathrm{t}$ 换成 $- \frac{1}{\mathrm{t}}$ 代入刚才得到的表达式里即可得到 $\left| \mathrm{{AC}}\right|  = \frac{2\left| {-\frac{1}{\mathrm{t}}}\right| \sqrt{1 + \frac{1}{{\mathrm{t}}^{2}}}}{\frac{1}{{\mathrm{t}}^{2}} + \mathrm{u}} = \frac{2\sqrt{{\mathrm{t}}^{2} + 1}}{{\mathrm{{ut}}}^{2} + 1}$ (我们之前讲到的解点利用对称性只解一次的策略再一次派上了用场)。于是我们完成了 $\mathrm{t}$ 作为主自由度, $\mathrm{u}$ 作为椭圆背景的，对两个长度的控制过程。

现在我们把已知中的正弦条件给充要替换一下。很容易得到 $\frac{\left| \mathrm{{AB}}\right| }{\left| \mathrm{{AC}}\right| } = 2\sqrt{2}$ 。把刚才控制的结果代入这个牵制的过程，得到 $u$ 和 $t$ 之间的牵制关系: $\frac{2t}{{t}^{2} + u} = \frac{4\sqrt{2}}{u{t}^{2} + 1}$ ，再整理一下就得到 ${\mathrm{{ut}}}^{3} + \mathrm{t} = 2\sqrt{2}\left( {{\mathrm{t}}^{2} + \mathrm{u}}\right)$ ,以上全程都是在做充要替换,信息量没有任何流失。这时候,再简单理一下整个问题的过程。我们输入一个 $\mathrm{u} > 1$ 作为控制,利用上面这个方程式就输出 $\mathrm{t} > 0$ 的解。最后问的问题也等价于求 $\mathrm{t}$ 所能遍历到的范围(再转成真斜率即可)。所以就得先求 $\mathrm{t}$ 的范围。

把这个问题隔离出来, 思维焦点转移上来, 设问一下, 都能取到什么样的t 呀? 什么样的t 能被一个 $\mathrm{u} > 1$ 输出啊? 这样一思考,马上就看出来只需要找出所有的 $\mathrm{t}$ ,使得上述关于 $u$ 的方程有 $u > 1$ 的解。所以分个参: $u = \frac{2\sqrt{2}{t}^{2} - t}{{t}^{3} - 2\sqrt{2}}$ (注意讨论一下确保 $t \neq  \sqrt{2}$ ), 并且看出凡是满足 $\frac{2\sqrt{2}{t}^{2} - t}{{t}^{3} - 2\sqrt{2}} > 1$ 的 $t > 0$ 均可以。求解上述不等式即可。

整理得到 $\frac{{t}^{3} - 2\sqrt{2}{t}^{2} + t - 2\sqrt{2}}{{t}^{3} - 2\sqrt{2}} < 0 \Leftrightarrow  \frac{t - 2\sqrt{2}}{t - \sqrt{2}} < 0 \Leftrightarrow  \sqrt{2} < t < 2\sqrt{2}$ ,再和 $t > 0$ 取一下交 (必要的一步,不然逻辑不严谨) 即可。最后要输出的 $\mathrm{k} = \frac{1}{\mathrm{t}} \in  \left( {\frac{\sqrt{2}}{4},\frac{\sqrt{2}}{2}}\right)$ ,这就是最终答案。

这道题所体现的思维方式就是最简单的自由度分析, 要素网建立以及最基本的沿要素网通过初级的主自由度控制要素, 并不断充要替换出最后范围的问题, 思维非常纯正, 没有任何需要花费智商动脑子的地方, 全程所有过程十分自然, 对于这些我们一直以来反复强调思维方式熟悉的话这道题就是白送的。不过因为其涉及到的思维方式比较全面综合, 对于理解清楚题意的要求也比较高, 而且最后不等式的讨论也需要结合设问的思维方式去认知清楚自己现在完成了什么，接下来应该干什么。因此，题目对于思维的基本功的考察是很好的, 能够达到高考试卷难题的标准。作为单选第八题, 既体现了单选压轴的思维难度要求, 又使得每一步均非常符合正统的高中数学解题思维, 确保思维基本功扎实的学生能非常顺利的解出, 体现了出题人对题目难度把控的精准。同时, 本题的提问方法虽然在要素关系网的视野下看是很简单的, 但是对于思维比较混乱的同学而言, 这种提问会乍一看有些摸不着头脑, 从而又体现了提问的创新性。
