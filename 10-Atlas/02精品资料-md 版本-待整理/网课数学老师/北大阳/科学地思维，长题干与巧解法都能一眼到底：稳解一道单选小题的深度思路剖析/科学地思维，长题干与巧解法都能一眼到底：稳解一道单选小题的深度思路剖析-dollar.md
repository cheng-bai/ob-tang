## 题目:

2022 年 12 月 4 日 20 点 10 分，神舟十四号返回舱顺利着陆，人们清楚全面地看到了神舟十四号返回舱成功着陆的直播盛况。根据搜救和直播的需要，在预设着陆场的某个竖直平面内设置了两个固定拍摄机位 $\mathrm{A},\mathrm{B}$ 和一个移动拍摄机位 $\mathrm{C}$ 。根据当时的气候与地理特征， 点 $\mathrm{C}$ 在抛物线 $\Gamma  : \mathrm{y} = \frac{1}{36}{\mathrm{x}}^{2}$ 上(直线 $\mathrm{y} = 0$ 与地平线重合, $\mathrm{y}$ 轴垂直于水平面,单位: 十米,下同),且 $\mathrm{C}$ 的横坐标 ${\mathrm{x}}_{\mathrm{C}} > 6\sqrt{2}$ 。已知 $\mathrm{A}$ 的坐标为 $\left( {-{36},2}\right)$ ,设点 $\mathrm{D}\left( {0, - 2}\right)$ ,线段 AC, DC 分别交 $\Gamma$ 于点 $\mathrm{M},\mathrm{N},\mathrm{B}$ 在线段 $\mathrm{{MN}}$ 上。则两固定机位 $\mathrm{A},\mathrm{B}$ 间的距离为( )

A. ${360}\mathrm{m}$ B. ${340}\mathrm{m}$ C. ${320}\mathrm{m}$ D. ${270}\mathrm{\;m}$

这道题目的题干属于现实情景类, 而且比较长。但是即便是这样, 动静视角总览全局, 以及分析要素结构的思维方式还是不变的, 无非是以前直接看已知构建要素关系网, 与现在理解题意构建要素关系网的区别。情景是安放拍摄机位, 翻译成要素关系就是研究机位的位置 (点要素)。首先是两个静止机位,也就是静态点 $\mathrm{A},\mathrm{B}$ ,然后一个单自由度动点 $\mathrm{C}$ 在 $\Gamma  : \mathrm{y} = \frac{1}{36}{\mathrm{x}}^{2}$ 的一部分上运动。然后要素关系网自然地往前走, $\mathrm{{AC}},\mathrm{{DC}}$ 分别交 $\Gamma$ 于点 $\mathrm{M},\mathrm{N}$ ,这一句全都是控制关系。所以目前关系网就是: $\mathrm{A},\mathrm{B}$ 静态, $\mathrm{A}$ 位置已知, $\mathrm{B}$ 具体位置不明 (但确定), 然后 C 带有一个自由度。因为题目并没有告诉我们 B 在哪, 再加上刚才的要素关系网它也没出现, 所以暂时放在一边。于是题目的状况就是一个单自由度的要素关系网再加一个不明位置的定点 B 。

但是这时，唐突出现了一个条件:B 在线段 MN 上。这是什么情况呢? 首先，目前题目并没有告诉我们 B 的具体位置，所以还在迷雾之中。而且画了要素关系网就知道，关系网到目前为止 B 都没有参与进来。所以这个已知条件是要告诉我们 B 到底在哪的。注意 C 作为移动机位, 这个自由度是肯定有的, 不可能被解出 (不然就是题目出错了)。那么, 线段 MN 就一定是一个单自由度系统控制的要素。所以题目的含义就很明确了:这个单自由度控制的 MN 需要过定点，而这个定点就是我们要找的 B ! 这就告诉我们:这道题其实正是我们重点讲过的不变量问题, 题目会给你一个自由度非 0 的系统, 然后开上帝视角的天眼告诉你这个系统可以输出一个不变量, 让你验证并计算这个不变量。因此, 这个现实情景的长题干就被我们利用动静视角与要素关系网的思维给转化成一个纯数学的问题了。

转化完的问题如下: 单自由度动点 $\mathrm{C}$ 在 $\Gamma  : \mathrm{y} = \frac{1}{36}{\mathrm{x}}^{2}\left( {\mathrm{x} > 6\sqrt{2}}\right)$ 上运动,点 $\mathrm{A}\left( {-{36},2}\right)$ , 点 $\mathrm{D}\left( {0, - 2}\right)$ 。线段 $\mathrm{{AC}},\mathrm{{DC}}$ 分别交 $\Gamma$ 于点 $\mathrm{M},\mathrm{N}$ ,证明线段 $\mathrm{{MN}}$ 过定点,并求出该定点。 下面自然就是依据要素关系网，选择合适的元问题展开计算了。要研究 MN ，自然就是要研究 M, N 两点, 而这两点又是抛物线上动点分别与两个不同的定点连线, 再与抛物线交出来的点。这一个控制关系能对应什么元问题呢? 太明显了, 不就是解点法吗? 我在 “数学的干货”视频列表里讲解点的时候，说的元问题和这个几乎一字不差了，但凡认真看过视频的同学，都不可能想不到。做两次解点把 $\mathrm{M},\mathrm{N}$ 控制出来，然后看定点即可。预判一下计算量大吗? 抛物线解点本来就比椭圆双曲线简单, 对好开口方向写直线列方程 (上下开口列 $\mathrm{y} = \mathrm{{kx}} + \mathrm{b}$ ，左右开口列 $\mathrm{x} = \mathrm{{ty}} + \mathrm{m}$ )计算即可，解两次点也可以接受。然后写个直线方程也轻而易举，判定可以算！

那就开算呗! 先把主自由度给 $\mathrm{C}\left( {\mathrm{t},\frac{1}{36}{\mathrm{t}}^{2}}\right)$ ,注意定义域 $\mathrm{t} > 6\sqrt{2}$ (设主自由度看定义域也强调多次了) 。于是写出 $\mathrm{{CD}}$ 方程 $\mathrm{y} = \frac{{\mathrm{t}}^{2} + {72}}{{36}\mathrm{t}}\mathrm{x} - 2$ 。不想直接代入这么长串的斜率, 可以用一下多级计算: 设 ${\mathrm{k}}_{1} = \frac{{\mathrm{t}}^{2} + {72}}{{36}\mathrm{t}}$ ,于是与抛物线联立得到 $\frac{1}{36}{\mathrm{x}}^{2} - {\mathrm{k}}_{1}\mathrm{x} + 2 = 0$ ,于是用韦达定理得 $t \cdot  {x}_{N} = {72}$ ,这就控制了 ${x}_{N}$ ,进而得 $N\left( {\frac{72}{t},\frac{144}{{t}^{2}}}\right)$ 。同理,可以算出 ${AC}$ 的斜率为 ${\mathrm{k}}_{2} = \frac{{\mathrm{t}}^{2} - {72}}{{36}\left( {\mathrm{t} + {36}}\right) }$ ,于是其方程为 $\mathrm{y} - 2 = {\mathrm{k}}_{2}\left( {\mathrm{x} + {36}}\right)$ (当然用 $\mathrm{C}$ 写点斜式也无所谓, 多级计算最后还原回去一定被 $\mathrm{t}$ 控制),代入抛物线并整理得 $\frac{1}{36}{\mathrm{x}}^{2} - {\mathrm{k}}_{2}\mathrm{x} - {36}{\mathrm{k}}_{2} - 2 = 0$ , 于是 $\mathrm{t} + {\mathrm{x}}_{\mathrm{M}} = {36}{\mathrm{k}}_{2} = \frac{{\mathrm{t}}^{2} - {72}}{\mathrm{t} + {36}}$ ,进而得 $\mathrm{M}\left( {\frac{-{36}\left( {\mathrm{t} + 2}\right) }{\mathrm{t} + {36}},\frac{{36}{\left( \mathrm{t} + 2\right) }^{2}}{{\left( \mathrm{t} + {36}\right) }^{2}}}\right)$ 。

我们已经实现了: 用单自由度来控制 $\mathrm{M},\mathrm{N}$ 。接下来有两种方法可选,对应于不同的要素网理解。单自由度问题如何处理? 一种处理方式是控制走到底: 用 $\mathrm{t}$ 表达线段 $\mathrm{{MN}}$ 的方程。因此计算 ${\mathrm{k}}_{\mathrm{{MN}}} = \frac{\frac{{\left( \mathrm{t} + 2\right) }^{2}}{{\left( \mathrm{t} + {36}\right) }^{2}} - \frac{4}{{\mathrm{t}}^{2}}}{\frac{-\left( {\mathrm{t} + 2}\right) }{\mathrm{t} + {36}} - \frac{2}{\mathrm{t}}} = \frac{2}{\mathrm{t}} - \frac{\mathrm{t} + 2}{\mathrm{t} + {36}}$ ,于是直线 $\mathrm{{MN}}$ 的方程就可以直接写出来: $y - \frac{144}{{t}^{2}} = \left( {\frac{2}{t} - \frac{t + 2}{t + {36}}}\right) \left( {x - \frac{72}{t}}\right)  \Leftrightarrow  y = \frac{{72} - {t}^{2}}{t\left( {t + {36}}\right) }x + \frac{{72}\left( {t + 2}\right) }{t\left( {t + {36}}\right) }$ 。注意线段交点一定是直线交点, 所以我们先看直线交点。这个方程能过某个定点吗? 那就要注意观察了, 我们希望 $x$ 取某个数值的时候,分子的 $- x{t}^{2} + {72t} + {72}\left( {2 + x}\right)$ 和 $t\left( {t + {36}}\right)$ 可以彻底约分,那一个直接感受就是得让前者的常数项 ${72}\left( {2 + x}\right)  = 0 \Leftrightarrow  x =  - 2$ ,这时分子变为 ${2t}\left( {t + {36}}\right)$ ,约掉分母变成常数 2,所以我们就看出来了它的确过定点 $\left( {-2,2}\right)$ 。

本题还有另一种处理思维,这是另一种对要素网的理解。 $\mathrm{M},\mathrm{N}$ 的坐标我们已经用单自由度刻画清楚了, 而我们要研究直线 MN 这一对象, 我们刻画的是它与抛物线的交点坐标。直线, 与抛物线交点坐标, 这一控制的元问题是什么? 当然是联立韦达定理了! 注意一般直线联立的时候,比如我们设 $\mathrm{{MN}} : \mathrm{y} = \mathrm{{kx}} + \mathrm{b}$ ,因为直线方程含有两个主自由度,因此需要韦达定理的两个式子才能把 $\mathrm{k},\mathrm{\;b}$ 的信息量全部刻画出来。比如这里联立 $\mathrm{{MN}}$ 与抛物线得 $\frac{1}{36}{x}^{2} - {kx} - b = 0$ ,可知 ${x}_{M} + {x}_{N} = {36k},{x}_{M}{x}_{N} =  - {36b}$ 。这两个式子都是必要的,丢失任何一个信息量都会流失, 而不像解点那样任取一个韦达定理式子就能完成全部控制 (因为解点法是已知了一个交点的信息了,信息量更多)。这样的话,我们希望用 $\mathrm{M},\mathrm{N}$ 的坐标之间的关系来等价刻画 $\mathrm{k},\mathrm{b}$ 之间的关系。因为 $\mathrm{t}$ 是一个自由度,所以如果想要充要替换成两个参数之间的式子,就应当是一个式子。获取方法就是消元法: 利用 ${\mathrm{x}}_{\mathrm{N}} = \frac{72}{\mathrm{t}}$ 与 ${x}_{M} = \frac{-{36}\left( {t + 2}\right) }{t + {36}}$ 消去 $t$ 得 ${x}_{M} = \frac{-{36}\left( {\frac{72}{{x}_{N}} + 2}\right) }{\frac{72}{{x}_{N}} + {36}} \Leftrightarrow  2\left( {{x}_{M} + {x}_{N}}\right)  + {x}_{M}{x}_{N} =  - {72}$ ,后面这个式子牵制了 ${\mathrm{X}}_{\mathrm{M}},{\mathrm{X}}_{\mathrm{N}}$ ,也就是牵制了 $\mathrm{k},\mathrm{b}$ (两个参数) 之间的关系,于是自由度是 1,与原题总自由度守恒,因此信息量与原题保持一致。现在我们取了 $\mathrm{k},\mathrm{b}$ 作为主自由度来刻画直线 $\mathrm{{MN}}$ 的信息了,用韦达定理可以实现 $\mathrm{k},\mathrm{b}$ 对 ${\mathrm{x}}_{\mathrm{M}},{\mathrm{x}}_{\mathrm{N}}$ 的控制,接下来只需要把牵制关系转回 $\mathrm{k},\mathrm{\;b}$ 之间的式子,即可实现单自由度对直线 $\mathrm{{MN}}$ 的控制。代入韦达定理式得到 $- 2\mathrm{k} + \mathrm{b} = 2$ 。 直线本身是 $\mathrm{y} = \mathrm{{kx}} + \mathrm{b}$ ,对照一下即可看出它过定点 $\left( {-2,2}\right)$ 。

不管用哪种方法，定点都算是找出来了，当然保险起见，大家还需要验证一下它确实是在线段 MN 上, 这个留作练习。但定点肯定只有一个, 只要题目没出错, 那么 B 就一定是 (-2,2) 。最后算出 A, B 之间距离为 $\left| {-{36} - \left( {-2}\right) }\right|  = {34}$ ,单位为十米,于是就是 ${340}\mathrm{\;m}$ 。

做个总结。本题仍然是使用 up 主反复讲过的思维方式, 按部就班做题, 碰到该用什么思维了就直接用啥思维，都不带拐弯的。先把问题的要素关系网建构清楚，然后把要素关

系网对应的元问题找出来, 加以预判可计算之后直接开算。比较无脑的解法就是第一条路, 直接用 t 一路控制到底, 利用分子分母的多项式可以互相约分的特性把定点找出来。第二条路则是对于当前要素关系: “坐标 ${\mathrm{x}}_{\mathrm{M}},{\mathrm{x}}_{\mathrm{N}}$ 控制直线” 的另一种理解,直线也可以反过来控制坐标, 这是一个双向控制。遇到可以双向控制的要素的时候, 颠倒要素网常常是一种可选的办法，要素网可以颠倒这件事情我们也早在经验分享第四期就给大家提到了。不管选用什么样的要素关系网，只要控制减牵制得到的总自由度不变，那么系统的运动幅度也就不变, 信息量也就维持了稳定。而有时根据当前要素网的内容(元问题)的认识, 进行适当的要素网改写, 常常能极大地改变计算的形式。如果预判某一步的计算量过于不稳定, 像本题第二条路这样, 重新选取主自由度改写一下要素关系网, 也是一条可选的, 并且时不时就能带来惊喜的路径。可以看到, 本题第二条路径的计算量明显低于第一条路径, 就说明了这一点。
